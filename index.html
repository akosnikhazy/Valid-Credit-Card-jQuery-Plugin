This demo starts with a valid credit card number. Change anything in it and most likely you will get a false one. I do not ask you to trust in an input field on the internet. This saves nothing, but do not trust me, trust the code.

It uses the Hans Peter Luhn algorithm to check the card number. How it works is you double the value of every second digit; if the doubled value is bigger than 9 you add up the digits of that number and at the end you add up all the digits you made this way with the digits you didn't touch. If the result is dividable by ten, it is a valid credit card number.
